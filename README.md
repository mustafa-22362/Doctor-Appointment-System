# Doctor Appointment System

A full-stack web application that allows patients to book appointments with doctors.

## Features

- User authentication
- Doctor management
- Appointment booking
- Patient dashboard
- Doctor dashboard
- Contact system

## Technologies Used

### Frontend
- React.js
- CSS
- JavaScript

### Backend
- Node.js
- Express.js
- MongoDB

## Installation

Clone the repository:

git clone https://github.com/mustafa-22362/Doctor-Appointment-System.git

Install dependencies:

Frontend:
npm install

Backend:
npm install

Run project:

npm start
